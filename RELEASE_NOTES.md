# FloatingTranslator v1.0.0

桌面划词翻译器 — 选中文本，悬浮窗弹出翻译。支持免费在线、LLM API、本地模型三种引擎。

## Linux

需安装 `xclip`：

```bash
sudo apt install xclip
tar -xzf FloatingTranslator-v1.0.0-linux-x86_64.tar.gz && ./FloatingTranslator/FloatingTranslator
```

## Windows

解压后双击 `FloatingTranslator.exe` 即可运行。

## 校验

```bash
sha256sum -c FloatingTranslator-v1.0.0-linux-x86_64.tar.gz.sha256
```
